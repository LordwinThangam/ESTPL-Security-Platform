Comprehensive collection of industry-standard security tools with Windows deployment instructions. All tools are FREE and open-source.

Penetration Testing Tools

Metasploit Framework

World's most used penetration testing framework with extensive exploit database and payload generation capabilities.

Penetration Testing Windows/Linux/Mac
Windows Setup:
  1. Download installer from metasploit.com
  2. Run installer as Administrator
  3. Open Command Prompt: msfconsole
  4. Update: msfupdate
Download Docs
Burp Suite Community

Leading toolkit for web application security testing with powerful proxy and scanning capabilities.

Web Security Windows/Linux/Mac
Windows Setup:
  1. Download from portswigger.net/burp/communitydownload
  2. Install Java JDK if needed
  3. Run installer, launch Burp Suite
  4. Configure browser proxy to 127.0.0.1:8080
Download Docs
OWASP ZAP

Free open-source web app security scanner with automated scanning and manual testing tools.

Web Scanner Windows/Linux/Mac
Windows Setup:
  1. Download from zaproxy.org/download
  2. Run installer (.exe)
  3. Launch ZAP, configure browser proxy
  4. Start Quick Scan or Manual Explore
Download Docs
SQLMap

Automated SQL injection and database takeover tool supporting multiple database systems.

SQL Injection Windows/Linux/Mac
Windows Setup:
  1. Install Python 3.x from python.org
  2. Download: git clone https://github.com/sqlmapproject/sqlmap.git
  3. Navigate to folder: cd sqlmap
  4. Run: python sqlmap.py -h
Download Docs

Network Analysis Tools

Nmap Integrated

Network discovery and security auditing tool. Already integrated in ESTPL platform.

Network Scanner Windows/Linux/Mac
Windows Setup:
  1. Download from nmap.org/download.html
  2. Run installer (includes Npcap)
  3. Open Command Prompt: nmap -h
  4. Basic scan: nmap 192.168.1.1
Download Docs
Wireshark

World's foremost network protocol analyzer for packet capture and deep inspection.

Packet Analysis Windows/Linux/Mac
Windows Setup:
  1. Download from wireshark.org/download.html
  2. Run installer (includes Npcap)
  3. Launch Wireshark, select network interface
  4. Click "Start capturing packets"
Download Docs
Aircrack-ng

Complete suite for wireless network security assessment and password cracking.

Wireless Security Windows/Linux
Windows Setup:
  1. Download from aircrack-ng.org/downloads.html
  2. Requires USB WiFi adapter with monitor mode
  3. Install drivers and Aircrack-ng suite
  4. Run: aircrack-ng --help
Download Docs

World's fastest password recovery tool with GPU acceleration support.

Password Cracking Windows/Linux/Mac

Web Security Scanners

Nikto

Open-source web server scanner for dangerous files, outdated versions, and vulnerabilities.

Web Scanner Windows/Linux/Mac
Windows Setup:
  1. Install Perl from strawberryperl.com
  2. Download: git clone https://github.com/sullo/nikto.git
  3. Navigate to program folder: cd nikto/program
  4. Run: perl nikto.pl -h target.com
Download Docs
Gobuster

Fast directory/file and DNS brute-forcing tool written in Go for web enumeration.

Directory Brute-Force Windows/Linux/Mac
Windows Setup:
  1. Install Go from golang.org
  2. Run: go install github.com/OJ/gobuster/v3@latest
  3. Add to PATH: %USERPROFILE%\go\bin
  4. Test: gobuster version
Download Docs
Drishti

Comprehensive security assessment framework for vulnerability scanning and pentesting.

Security Framework Windows/Linux/Mac
Windows Setup:
  1. Install Python 3.x from python.org
  2. Download: git clone https://github.com/h4cklife/drishti.git
  3. Install dependencies: pip install -r requirements.txt
  4. Run: python drishti.py
Download Docs
